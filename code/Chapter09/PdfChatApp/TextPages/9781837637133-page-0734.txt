Exercise 15.3 – Practice building an IndexedDB interop service Browser local and session storage its okay for storing small amounts of data, but if you need a more robust and capable storage in the browser, then you can use the IndexedDB API. Create a Blazor service named IndexedDbService, with a JavaScript module for interop, with the IndexedDB API, and then use it to cache the employees. You can learn more about the methods of the window.indexedDB object at the following link: https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API.
