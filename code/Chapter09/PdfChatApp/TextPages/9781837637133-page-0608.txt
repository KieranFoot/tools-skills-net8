In the Services folder, in GreeterService.cs, note that it inherits from a class named GreeterBase and it asynchronously implements the Greeter service contract by having a SayHello method that accepts a HelloRequest input parameter and returns a HelloReply, as shown in the following code: using Grpc.Core; using Northwind.Grpc.Service namespace Northwind.Grpc.Service.Services { public class GreeterService : Greeter.GreeterBase { private readonly ILogger<GreeterService> _logger; public GreeterService(ILogger<GreeterService> logger) { _logger = logger; } public override Task<HelloReply> SayHello( HelloRequest request, ServerCallContext context) { return Task.FromResult(new HelloReply { Message = "Hello " + request.Name }); } } } 6.
