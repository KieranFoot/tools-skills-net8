In Visual Studio 2022, right-click the Northwind.Maui.Client project folder, choose Add | New Item... or press Ctrl + Shift + A, select .NET MAUI in the template types tree, select .NET MAUI ContentPage (XAML), enter the name SettingsPage, and click Add. Visual Studio Code and JetBrains Rider do not have project item templates for MAUI. You can create this item using the CLI, as shown in the following command: dotnet new maui-page-xaml --name SettingsPage.xaml 7.
