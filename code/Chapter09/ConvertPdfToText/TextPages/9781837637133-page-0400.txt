Chapter 8 367 Summary In this chapter, you learned how to: • Build a web service that implements the REST architectural style using minimal APIs. • Relax the same-origin security policy for specified domains and ports using CORS. • Implement two different rate-limiting packages to prevent denial of service attacks. • Secure services using JWT bearer authorization.
